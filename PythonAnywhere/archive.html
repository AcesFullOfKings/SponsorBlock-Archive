About This Archive

This archive contains compressed daily snapshots of the SponsorBlock database, converted from the raw CSV exports into space-efficient SQLite databases. The data is split across two types of file:

Daily files contain only the votes and views for every segment on that day. To get complete segment data (video ID, timing, category, user, submission date, etc.), you'll also need the static database, which is a master lookup table that maps each segment's short integer ID to its full metadata. Download both, and you can join them on the segment ID to get the full picture for any given day.

A Python utility script, helper_functions.py, is available to help you query these databases. It provides functions for looking up segments, users, and available dates. Just update the database path variables at the top of the file to point to your local copies.

The monthly archives (.7z) are the original, unmodified CSV files as exported by SBServer (the SponsorBlock server). These are very large and are provided for completeness.

Static Database

Master lookup table containing user IDs and segment metadata (video ID, timing, category, submission date). This file grows over time as new segments are submitted. You only need one copy of this file.
